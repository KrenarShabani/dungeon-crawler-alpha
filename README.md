# Dungeon Crawler — alpha

A squad-based real-time dungeon crawler. Take a party of adventurers down 15 floors:
fight, loot, hire, extract — permadeath is real, ghosts are realer.

## ▶ Play in your browser
**[Click here to play](https://krenarshabani.github.io/dungeon-crawler-alpha/)** — no install (first load streams ~330 MB, give it a minute).
